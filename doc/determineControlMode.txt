# @iseeberg79/solar-forecast-nodes

## DetermineControlMode Node

Dieser Node bestimmt den Steuerungsmodus basierend auf verschiedenen Faktoren.

### Konfigurationsparameter

- **Name**: Der Name des Nodes.
- **Minimum PV Required**: Minimum erforderliche PV-Leistung.
- **Minimum Price Deviation**: Mindestpreisdifferenz.
- **Minimum Price Difference**: Mindestpreisdifferenz.
- **Price Limit**: Preisgrenze.
- **Average Price**: Durchschnittspreis.

### Eingänge und Ausgänge

- **Inputs**: 1
- **Outputs**: 1

### Verwendung

1. Fügen Sie den Node zu Ihrem Flow hinzu.
2. Konfigurieren Sie die erforderlichen Parameter (Name, Minimum PV Required, Minimum Price Deviation, Minimum Price Difference, Price Limit, Average Price).
3. Verbinden Sie den Node mit anderen Nodes in Ihrem Flow.

### Beispiel

```json
[
    {
        "id": "c6ed6e04461115c3",
        "type": "@iseeberg79/DetermineControlMode",
        "name": "DetermineControlMode",
        "minPvRequired": 16000,
        "minPriceDeviation": 6,
        "minPriceDifference": 15,
        "priceLimit": 0.35,
        "avg": 0.24,
        "wires": [
            ["90e1013fda31b0e7"]
        ]
    }
]
