# @iseeberg79/solar-forecast-nodes

## EvaluateGridEnergyPrices Node

Dieser Node wertet die Strompreise auf Basis der bereitgestellten Daten aus.

### Konfigurationsparameter

- **Name**: Der Name des Nodes.
- **BZN**: Gebietsnetz (z.B. DE-LU).
- **URL**: Die URL für die Datenanfrage (z.B. `https://api.energy-charts.info/price`).
- **Charges**: Zuzüglich berechnete Gebühren.
- **Tax**: Steuerprozentsatz (z.B. 19 für 19%).

### Eingänge und Ausgänge

- **Inputs**: 1
- **Outputs**: 1

### Verwendung

1. Fügen Sie den Node zu Ihrem Flow hinzu.
2. Konfigurieren Sie die erforderlichen Parameter (Name, BZN, URL, Charges, Tax).
3. Verbinden Sie den Node mit anderen Nodes in Ihrem Flow.

### Beispiel

```json
[
    {
        "id": "6dcfc7b3890db5ed",
        "type": "@iseeberg79/EvaluateGridEnergyPrices",
        "name": "EvaluateGridEnergyPrices",
        "bzn": "DE-LU",
        "url": "https://api.energy-charts.info/price",
        "charges": 12.945,
        "tax": 19,
        "wires": [
            ["3cf2622713ccb5b6"]
        ]
    }
]
